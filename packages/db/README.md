# @dc-beauty/db

Database package with Prisma ORM.

## Usage

```bash
# Generate Prisma client
pnpm generate

# Run migrations
pnpm migrate

# Open Prisma Studio
pnpm studio
```

## Development

1. Update `prisma/schema.prisma`
2. Run `pnpm generate` to update client
3. Run `pnpm migrate` to apply changes
